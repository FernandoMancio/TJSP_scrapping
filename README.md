# TJSP_scrapping

> Descrição: Python code to extract procedural information from the lawsuit number. They were divided due to necessity during creation.
> Contexto/Motivação: Para construir uma base de dados para análise. Já que não possuia loging como advogado e nem  informações para a análise.
> Dados: Foram obtidos atraves do site do TJSP: https://esaj.tjsp.jus.br/cpopg/open.do
>	Metodologia/Solução: Através de scripts python, realizei extrações as quais rodavam em segundo plano no meu pc e eu continuei minhas atividades enquanto ele rodava.
>	Resultados/Insights: Com isto, consegui ter uma ideia dos objetos pedidos na ação, se haviam outras pessoas na ação, KPI de vitorias com determinado juiz e advogados. Entre outros pontos cruciais.
>	Como Rodar: Ao salvar o idle do python na máquina, deve-se inserir os números dos processos a serem consultados, em: numeros_processos e definir o path em: caminho_planilha. E então rodar o script.
>	Tecnologias: Linguagem Python.
